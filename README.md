# saludtech-2025

Uniandes - Diseño y construcción de soluciones no monolíticas - 2025

## Estructura del proyecto

```
saludtech-2025/
├── .gitignore                          # Archivo que especifica qué archivos y directorios deben ser ignorados por Git.
├── .gitpod.Dockerfile                  # Archivo de configuración para Gitpod, especifica el entorno de desarrollo.
├── .gitpod.yml                         # Archivo de configuración para Gitpod, define tareas y configuraciones adicionales.
├── .java-version                       # Archivo que especifica la versión de Java a utilizar.
├── .servicecutter.yml                  # Archivo de configuración para Service Cutter.
├── build.gradle                        # Script de construcción de Gradle para el proyecto.
├── gitpod.yml                          #  Archivo de configuración para Gitpod.
├── gradle/                             # Directorio que contiene archivos relacionados con Gradle.
├── gradle.properties                   # Archivo de propiedades de Gradle.
├── gradlew                             # Script para ejecutar Gradle en sistemas Unix.
├── gradlew.bat                         # Script para ejecutar Gradle en sistemas Windows.
├── README.md                           # Archivo de documentación del proyecto.
├── settings.gradle                     # Archivo de configuración de Gradle para definir los proyectos incluidos.
├── entregas/                           # Directorio de documentación de entregas.
├── src/                                # Directorio principal del código fuente.
│   ├── main/                           # Contiene el código fuente principal.
│   │   └── salud-tech-los-alpes/       # Contiene los archivos de modelado específicos del proyecto.
│   │       ├── ContextoASIS.cml        # Archivo de modelado del contexto actual.
│   │       ├── ContextoTOBE.cml        # Archivo de modelado del contexto futuro.
│   │       └── Dominios.cml            # Archivo de modelado de dominios.
├── .gradle/                            # Directorio que contiene archivos y datos generados por Gradle.
│   ├── 7.4.2/                          # Archivos específicos de la versión de Gradle.
│   └── vcs-1/                          # Archivos de metadatos del sistema de control de versiones.
```
